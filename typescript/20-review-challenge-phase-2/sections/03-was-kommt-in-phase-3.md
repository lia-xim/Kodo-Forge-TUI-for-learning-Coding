# Was kommt in Phase 3?

> Geschaetzte Lesezeit: **5 Minuten**

## Phase 3: Advanced TypeScript & Frameworks

Phase 3 baut auf deinem soliden Type-System-Wissen auf und verbindet es
mit realen Frameworks und fortgeschrittenen Patterns.

---

## Vorschau der Themen

### Classes & OOP (L21)
- Klassen mit Access Modifiers (public, private, protected)
- Abstract Classes und Interfaces implementieren
- Generische Klassen
- Mixins und Composition vs Inheritance

### Advanced Generics & Type Gymnastics (L22)
- Variadic Tuple Types
- Template Literal Types + Mapped Types Kombination
- Recursive Type Challenges (Type-Level Programmierung)
- Reale Type Challenges von type-challenges

### Decorators (L23)
- Stage 3 Decorators (TC39)
- Class, Method, Property, Parameter Decorators
- Decorator Factories
- Metadata und Reflection

### Framework-Integration (L24-L30)
- **Angular:** Component Types, RxJS Generics, Dependency Injection
- **React:** Component Props, Hooks Types, Context API
- **Next.js:** Server Components, API Routes, Middleware Types
- **State Management:** Redux/NgRx/Zustand Typ-Patterns
