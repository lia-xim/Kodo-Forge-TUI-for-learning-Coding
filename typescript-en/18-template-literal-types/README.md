# Lesson 18: Template Literal Types

> **Prerequisite:** Lesson 17 (Conditional Types) completed.
> **Goal:** Master string manipulation at the type level — from simple concatenation to type-safe event systems and route parsers.
> **Core question of this lesson:** How do I manipulate strings at compile time for maximum type safety?
> **Total duration:** ~50 minutes (5 sections of ~10 minutes each)
